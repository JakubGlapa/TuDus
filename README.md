# TuDus

<br />

## Description
Creating keyboard-first ToDo windows program with notion-like syntax. <br />
new TuDu -> [ ]/[] <br />
done TuDu -> [x] <br />
add due date to TuDu -> @ <br />
add tag to TuDu -> # <br />
